## Get museum hours

Get upcoming museum operating hours

### Query Parameters

<Property required={false} deprecated={false} name={"startDate"} type={"string"}>

The starting date to retrieve future operating hours from. Defaults to today's date.

<span>Example: `"2023-02-23"`</span>

<span>Format: `"date"`</span>

</Property>

<Property required={false} deprecated={false} name={"page"} type={"integer"}>

The page number to retrieve.

<span>Example: `2`</span>

<span>Default: `1`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The number of days per page.

<span>Example: `15`</span>

<span>Default: `10`</span>

<span>Maximum: `30`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200`       | Success     |
| `400`       | Bad request |
| `404`       | Not found   |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X GET "https://api.fake-museum-example.com/museum-hours?startDate=2023-02-23&page=2&limit=15"
```

<Tabs items={["200"]}>

<Tab value={"200"}>

```json title="Example Response"
[
  {
    "date": "02-02-2022",
    "timeOpen": "09:00",
    "timeClose": "18:00"
  }
]
```

</Tab>

</Tabs>

</APIExample>

</API>

## List special events

### Query Parameters

<Property required={false} deprecated={false} name={"startDate"} type={"string"}>

The starting date to retrieve future operating hours from. Defaults to today's date.

<span>Example: `"2023-02-23"`</span>

<span>Format: `"date"`</span>

</Property>

<span>Example: `2`</span>

<span>Default: `1`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

The number of days per page.

<span>Example: `15`</span>

<span>Default: `10`</span>

<span>Maximum: `30`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200`       | Success     |
| `400`       | Bad request |
| `404`       | Not found   |

</APIInfo>

## Buy museum tickets

Purchase museum tickets for general entry or special events.

### Request Body

<Property required={true} deprecated={false} name={"ticketType"} type={"string"}>

Type of ticket being purchased. Use `general` for regular museum entry and `event` for tickets to special events.

<span>Example: `"event"`</span>

<span>Value in: `"event" | "general"`</span>

</Property>

<Property required={false} deprecated={false} name={"eventId"} type={"string"}>

Unique identifier for a special event. Required if purchasing tickets for the museum's special events.

<span>Example: `"3be6453c-03eb-4357-ae5a-984a0e574a54"`</span>

<span>Format: `"uuid"`</span>

</Property>

<Property required={true} deprecated={false} name={"ticketDate"} type={"string"}>

Date that the ticket is valid for.

<span>Example: `"2023-10-29"`</span>

<span>Format: `"date"`</span>

</Property>

<Property required={true} deprecated={false} name={"email"} type={"string"}>

Email address for ticket purchaser.

<span>Example: `"museum-lover@example.com"`</span>

<span>Format: `"email"`</span>

</Property>

<Property required={false} deprecated={false} name={"phone"} type={"string"}>

Phone number for the ticket purchaser (optional).

<span>Example: `"+1(234)-567-8910"`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200`       | Success     |
| `400`       | Bad request |
| `404`       | Not found   |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X POST "https://api.fake-museum-example.com/tickets?" \
  -d '{
  "ticketType": "event",
  "eventId": "3be6453c-03eb-4357-ae5a-984a0e574a54",
  "ticketDate": "2023-10-29",
  "email": "museum-lover@example.com",
  "phone": "+1(234)-567-8910"
}'
```

<Tabs items={["200"]}>

<Tab value={"200"}>

```json title="Example Response"
{
  "message": "Museum general entry ticket purchased",
  "eventName": "Pirate Coding Workshop",
  "ticketId": "a54a57ca-36f8-421b-a6b4-2e8f26858a4c",
  "ticketType": "event",
  "ticketDate": "2023-10-29",
  "confirmationCode": "ticket-event-a98c8f-7eb12"
}
```
