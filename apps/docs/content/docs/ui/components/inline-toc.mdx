## Usage

Pass TOC items to the component.

```mdx
import { InlineTOC } from 'next-docs-ui/components/inline-toc'

<InlineTOC items={toc} />
```

### Use in Pages

You can add inline TOC into every page.

```tsx
<MDXContent>
  ...
  <InlineTOC items={toc} />
  ...
</MDXContent>
```
