## Features

Built on the top of Next.js, Radix UI and **Next Docs Zeta**. Styled using
Tailwind CSS. Next Docs UI provides many features out of the box.

- Light/Dark Mode
- Full-text Search
- Table of Contents
- Built-in Components
- Internationalization (I18n)
- Code Syntax highlighting powered by Shiki

## Getting Started

Open in [CodeSandbox](https://githubbox.com/SonMooSans/next-docs-ui-template).

<Tabs items={["With CLI", "Manual"]}>

<Tab value="With CLI">

Create a new app with the CLI.

```bash
npx create-next-docs-app
# or
pnpx create-next-docs-app
# or
yarn create next-docs-app
```

> **Note:** this example uses Contentlayer, but you can use any source you want.

</Tab>

<Tab value="Manual">

Install required dependencies

```bash
npm install next-docs-ui next-docs-zeta shiki
```

<h3>Configuration</h3>

Next Docs UI doesn't provide static generation directly. Instead, it uses
[Contentlayer](https://www.contentlayer.dev).

```bash
npm install contentlayer next-contentlayer
```

1. **Configure Contentlayer**

```ts {4} title="contentlayer.config.ts"
|reference:../../examples/simple/contentlayer.config.ts|
```

2. **Add Contentlayer Plugin**

```js title="next.config.js"
|reference:../../examples/simple/next.config.js|
```

3. **Build Page Tree**

```ts title="app/source.ts"
|reference:../../examples/simple/app/source.ts|
```

Make sure you've generated documents correctly before moving to next step.

## Enjoy!

Now, Create your first mdx file in the docs folder.

```mdx title="/content/docs/index.mdx"
---
title: Hello World
---

## Yo what's up
```

Run the app in development mode and see http://localhost:3000/docs.

```bash
npm run dev
```
