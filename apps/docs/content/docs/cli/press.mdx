description: A light tool to view Markdown files.
---

## Introduction

Fumapress allows you to view Markdown files with Fumadocs, it is useful for reading skills files & notes.

Run the command under your project directory.

```npm
npx fumapress dir/to/content
```

### Configurations

For other configurations, you can install it locally:

```npm
npm install fumapress -D
```

Then init the config file:

```npm
npx fumapress init
```
