## Introduction

`fumadocs-preview` allows you to view Markdown files with Fumadocs, it is useful for reading skills files & notes.

Run the command under your project directory.

```npm
npx fumadocs-preview dir/to/content
```

### Configurations

For other configurations, you can install it locally:

```npm
npm install fumadocs-preview -D
```

Then init the config file:

```npm
npx fumadocs-preview init
```
