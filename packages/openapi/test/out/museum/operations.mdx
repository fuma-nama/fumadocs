## Get museum hours

Get upcoming museum operating hours

### Authorization

<Property name={"Authorization"} type={"Basic <token>"} required={true}>

In: `header`

</Property>

### Query Parameters

<Property name={"startDate"} type={"string"} required={false} deprecated={false}>

The starting date to retrieve future operating hours from. Defaults to today's date.

<span>Example:  `"2023-02-23"`</span>

<span>Format:  `"date"`</span>

</Property>

<Property name={"page"} type={"integer"} required={false} deprecated={false}>

The page number to retrieve.

<span>Example:  `2`</span>

<span>Default:  `1`</span>

</Property>

<Property name={"limit"} type={"integer"} required={false} deprecated={false}>

The number of days per page.

<span>Example:  `15`</span>

<span>Default:  `10`</span>

<span>Maximum:  `30`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success |
| `400` | Bad request |
| `404` | Not found |

</APIInfo>
