Security requirements are defined per operation in the AsyncAPI document. Supported schemes depend on the protocol and server configuration.

## Security Schemes

Check the **Authorization** section on each operation page for the applicable security schemes and scopes.

When connecting to a broker or gateway, configure credentials according to your environment (for example API keys, SASL, or TLS client certificates).
