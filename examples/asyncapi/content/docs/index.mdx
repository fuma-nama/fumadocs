Scalar Galaxy Events is an AsyncAPI 3.0 specification that documents real-time notifications and event-driven capabilities for the Scalar Galaxy API.

## Event-Driven Features

- **Planet Updates**: Get notified when planets are created, updated, or experience cosmic events
- **User Activity Streams**: Track user signups, authentication events, and profile changes
- **Celestial Body Monitoring**: Monitor changes to satellites, asteroids, and other celestial objects
- **System Notifications**: Receive alerts about system maintenance, rate limits, and security events

## Operations

AsyncAPI operations use `send` and `receive` actions instead of HTTP methods. Each operation references a channel and the messages it can process.

Browse the generated reference pages under **References** to explore channels, messages, bindings, and security schemes from the spec.
