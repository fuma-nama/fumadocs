## Content Source

The content lives in `content/docs` as MDX files, managed by TinaCMS. The `docs` collection is defined in `tina/config.ts`.

`lib/source.ts` creates a [dynamic source](https://fumadocs.dev/docs/headless/source-api/dynamic) with `createTinaCMSSource()`, which queries the Tina GraphQL content API.
